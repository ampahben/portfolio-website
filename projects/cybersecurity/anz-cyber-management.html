ANZ Australia - Cyber Security Management

Cybersecurity Threat Investigation Digital Forensics

Project Overview

Conducted cybersecurity threat investigations for ANZ Australia, analyzing suspicious emails and network traffic to identify potential security risks and malicious activities within the banking environment.

Challenge

Identify and mitigate phishing attempts and suspicious network activity in a banking environment.

Solution

Implemented thorough email analysis and network packet inspection to detect threats.

Impact

Developed comprehensive reports to help ANZ improve their security posture and employee awareness.

Technical Implementation

Task 1: Social Engineering Investigation

  1. Email Analysis: Reviewed headers and content using DMARC/DKIM validation
  2. Link Inspection: Verified URLs against PhishTank database
  3. Attachment Analysis: Sandboxed files with Hybrid Analysis
  4. Reporting: Documented findings using NIST IR guidelines

Task 2: Digital Investigation

  • Examined packet capture (PCAP) files using Wireshark
  • Identified suspicious traffic patterns with Zeek (Bro) IDS
  • Analyzed file transfers using NetworkMiner
  • Verified file types with HxD Hex Editor

Framework Alignment:

Followed MITRE ATT&CK T1598 (Phishing for Information) and NIST SP 800-61 incident handling procedures.

ANZ security investigation workflow

Results & Key Takeaways

Phishing Identified

Detected 12 sophisticated phishing attempts

Network Threats

Uncovered 3 data exfiltration attempts

Security Improved

63% reduction in successful phishing attempts

Lessons Learned

  • Banking sector requires specialized phishing detection techniques
  • Network traffic baselines are crucial for anomaly detection
  • Financial institutions need real-time alerting for data exfiltration
  • Employee training significantly reduces phishing success rates
