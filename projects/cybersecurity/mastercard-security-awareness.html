Mastercard - Security Awareness

Cybersecurity Phishing Defense

Project Overview

Designed and analyzed phishing simulation campaigns for Mastercard, creating realistic attack scenarios and developing targeted security awareness training based on simulation results to reduce organizational risk.

Challenge

Reduce phishing susceptibility across high-risk departments by 50%.

Solution

Implemented behavioral-based training with realistic simulations and immediate feedback.

Impact

Achieved 63% reduction in click-through rates across targeted departments.

Technical Implementation

Program Development

  • Designed curriculum around NIST SP 800-50 security awareness guidelines
  • Created templates using GoPhish and King Phisher
  • Tracked metrics with KnowBe4 dashboard integration
  • Mapped simulations to MITRE ATT&CK T1598 (Phishing for Information)

Training Methodology

  • Created interactive modules with Articulate 360
  • Developed department-specific training in Google Slides
  • Implemented PhishER for reported email analysis
  • Integrated with Slack for security tip delivery

Training Framework:

Program structure followed the NICE Framework (PM-TA-001) for security awareness training and the SANS Security Awareness Maturity Model for progression metrics.

Security awareness metrics dashboard

Results & Key Takeaways

HR Dept Improvement

75% reduction in phishing susceptibility

Overall Reduction

63% decrease in successful phishing attempts

Reporting Increase

42% more employees reporting suspicious emails

Lessons Learned

  • Department-specific training yields better results than generic programs
  • Simulations should evolve with the threat landscape quarterly
  • Positive reinforcement works better than fear-based training
  • Short, frequent training sessions outperform annual marathon sessions
